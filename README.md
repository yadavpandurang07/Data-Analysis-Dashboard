## Store Data Analysis using Excel

## Project Overview
This project analyzes store sales data to identify trends, product performance, and business insights using Microsoft Excel.

## Dataset used 
-<a herf="https://github.com/yadavpandurang07/Data-Analysis-Dashboard/blob/main/Copy%20of%20Store%20Data%20Analysis%5E%5E.xlsx">Dataset</a>

## Tools Used
- MS Excel
- Pivot Tables
- Charts & Graphs
- VLOOKUP / IF formulas
- Data Cleaning techniques

## Key Analysis
- Monthly and yearly sales trends
- Top-performing and low-performing products
- Category-wise sales distribution
- Revenue insights using pivot tables

## Key Insights
- Identified high-revenue products contributing most to total sales
- Found underperforming categories needing business improvement
- Created visual dashboards for decision-making
